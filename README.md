# ctf-rsa-breaker

This is a Python 2.7 script that can be used to solve RSA challenges in CTFs.

## Usage

```
ctf-rsa-breaker.py -p|--pubKey <public_key_file> -e|--encFile <encrypted_file> -o|--out <output_file>
```
